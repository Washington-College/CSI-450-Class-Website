# Placeholder for blog post images

This folder is for images used in blog posts.

To add an image to a blog post:

1. Save your image file in this folder (or in your post's folder)
2. Reference it in your `.qmd` file:
   ```markdown
   ![Image description](../images/your-image.jpg)
   ```

Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`, `.svg`
